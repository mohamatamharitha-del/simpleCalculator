Calculator Mobile Application

A simple and modern calculator application built for Android.
The app allows users to perform basic mathematical operations using both button input and voice input, along with a history feature.

Features

✔ Basic Arithmetic Operations ( + , − , × , ÷ )
✔ Clean & Minimal Dark Theme UI
✔ Voice Input (Microphone Support)
✔ Decimal Number Handling
✔ Delete / Backspace Functionality
✔ Calculation History
✔ Search & Delete History

Screens

Calculator Screen
History Screen

How to Use

1. Enter numbers using keypad or microphone
2. Select an operator
3. Press equals (=) to get result
4. Use C to clear
5. Open History Screen to view past calculations

Tech Stack

Platform: Android
Language: Kotlin
UI: Jetpack Compose
